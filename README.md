# plant_Disease
🌱 Plant Disease Detection
📖 Overview
This project leverages machine learning and deep learning techniques to identify plant diseases from leaf images. By automating disease detection, farmers and researchers can save time, reduce crop losses, and improve agricultural productivity.

🎯 Objectives
Build a robust image classification model for plant disease detection

Preprocess and augment plant leaf datasets for better accuracy

Evaluate performance using metrics like accuracy, precision, recall, and F1-score

Provide visualizations of predictions and misclassifications

🛠️ Tech Stack
Python (NumPy, Pandas, Matplotlib, Seaborn)

Deep Learning: TensorFlow / Keras, CNN architectures

Data Handling: OpenCV for image preprocessing

Deployment (optional): Streamlit / Flask for interactive web app

📂 Dataset
Publicly available plant leaf datasets (e.g., PlantVillage)

Images categorized by healthy vs. diseased leaves

Preprocessing includes resizing, normalization, and augmentation

🚀 Usage
Clone the repository

bash
git clone https://github.com/PrakharKishoreAI/plant_Disease.git
cd plant_Disease
Install dependencies

bash
pip install -r requirements.txt
Run the notebook or script to train the model

bash
python train.py
Test with sample images in the test_images/ folder

📊 Results
Achieved high accuracy on validation and test sets

Confusion matrix and classification report included

Visualizations of correctly and incorrectly classified samples
